# Midnight Project

Этот репозиторий содержит Java программы с графическим интерфейсом и настроен для автоматической синхронизации изменений.

## Java Программы

### 1. TodoApp.java - Менеджер задач
Простое приложение для управления списком задач с функциями:
- Добавление новых задач
- Удаление выбранных задач
- Очистка всего списка
- Красивый графический интерфейс

### 2. Calculator.java - Калькулятор
Полнофункциональный калькулятор с возможностями:
- Основные арифметические операции (+, -, ×, ÷)
- Работа с десятичными числами
- Смена знака числа (±)
- Вычисление процентов (%)
- Очистка (C)

## Запуск программ

### Автоматический запуск:
```powershell
.\run.ps1
```
или
```cmd
run.bat
```

### Ручная компиляция и запуск:
```bash
javac TodoApp.java
java TodoApp

javac Calculator.java
java Calculator
```

## Способы синхронизации

### 1. Разовая синхронизация
Для быстрой отправки изменений в репозиторий:
```powershell
.\sync.ps1
```

Или с собственным сообщением коммита:
```powershell
.\sync.ps1 -Message "Мое описание изменений"
```

### 2. Автоматическая синхронизация
Для постоянного мониторинга и автоматической отправки изменений:
```powershell
.\auto-sync.ps1
```

С настройкой интервала (в секундах):
```powershell
.\auto-sync.ps1 -IntervalSeconds 60
```

### 3. Ручная синхронизация
Стандартные Git команды:
```bash
git add .
git commit -m "Описание изменений"
git push origin main
```

## Первоначальная настройка

Если это первый коммит, выполните:
